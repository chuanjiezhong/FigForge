# FigForge 客户交付说明

## 交付内容

### 1. 应用安装包

根据客户的操作系统，提供对应的安装包：

- **macOS**: `FigForge-0.1.0.dmg`
- **Windows**: `FigForge Setup 0.1.0.exe`
- **Linux**: `FigForge-0.1.0.AppImage` 或 `figforge_0.1.0_amd64.deb`

### 2. 系统要求

#### 必需软件
- **R 环境**: R >= 4.0.0
  - macOS: 从 [CRAN](https://cran.r-project.org/) 下载安装
  - Windows: 从 [CRAN](https://cran.r-project.org/) 下载安装
  - Linux: 使用包管理器安装 `sudo apt install r-base` (Ubuntu/Debian)

#### 系统要求
- **macOS**: macOS 10.13 或更高版本
- **Windows**: Windows 10 或更高版本
- **Linux**: Ubuntu 18.04 或更高版本（或其他主流发行版）

### 3. 安装步骤

#### macOS
1. 下载 `FigForge-0.1.0.dmg` 文件
2. 双击打开 DMG 文件
3. 将 FigForge 拖拽到 Applications 文件夹
4. 首次运行可能需要在"系统偏好设置 > 安全性与隐私"中允许运行

#### Windows
1. 下载 `FigForge Setup 0.1.0.exe` 文件
2. 双击运行安装程序
3. 按照安装向导完成安装
4. 从开始菜单启动 FigForge

#### Linux
**AppImage 方式：**
1. 下载 `FigForge-0.1.0.AppImage` 文件
2. 添加执行权限：`chmod +x FigForge-0.1.0.AppImage`
3. 双击运行或命令行执行：`./FigForge-0.1.0.AppImage`

**Debian 包方式：**
1. 下载 `figforge_0.1.0_amd64.deb` 文件
2. 安装：`sudo dpkg -i figforge_0.1.0_amd64.deb`
3. 从应用程序菜单启动

### 4. 使用前准备

1. **安装 R 环境**
   - 确保 R 已正确安装
   - 验证安装：在终端运行 `R --version` 或 `Rscript --version`

2. **安装必要的 R 包**
   - 首次使用时，应用会提示安装必要的 R 包
   - 或手动安装：在 R 控制台运行 `install.packages(c("ggplot2", "其他需要的包"))`

### 5. 常见问题

#### 应用无法启动
- 检查系统要求是否满足
- 检查是否有杀毒软件拦截
- 查看错误日志（macOS: Console.app, Windows: 事件查看器）

#### R Pipeline 无法运行
- 确认 R 已正确安装并在系统 PATH 中
- 检查 R 包是否已安装
- 查看应用内的错误日志

#### 导出功能异常
- 确认有足够的磁盘空间
- 检查文件写入权限

### 6. 技术支持

如遇到问题，请提供以下信息：
- 操作系统版本
- FigForge 版本
- R 版本
- 错误截图或日志
- 操作步骤描述

### 7. 更新说明

应用更新时，请：
1. 备份当前项目文件
2. 卸载旧版本（可选，新版本通常可以覆盖安装）
3. 安装新版本
4. 恢复项目文件
